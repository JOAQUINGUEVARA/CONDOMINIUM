    ZONAS COMUNES

    Ley 428 de 1998

    Artículo 16°.- Áreas de recreación. Todas las Unidades Inmobiliarias Cerradas dispondrán proporcionalmente a su tamaño y al uso predominante de áreas comunes suficientes para actividades recreativas, culturales y deportivas. Tales exigencias podrán disminuirse cuando se garantice de otra manera el derecho a la práctica del deporte y a la recreación.

    La utilización de las áreas comunes de recreación se someterá a la reglamentación interna que expida la Asamblea de Copropietarios y la Junta Administradora de la Unidad Inmobiliaria Cerrada.

    Registro

    Sr. Residente si no está registrado, debe ingresar por la opción Usuario no Registado, el sistema le pedirá el token que se le proporcionó para acceder a la información del conjunto.

    Si no tiene token solicitelo con éste botón.

    Solicitar Token

    El sistema le va a pedir un mail y el interior y apartamento en que reside, si el mail no está registrado en la información de residentes del conjunto no podrá acceder.

    En éste caso debe enviar la información actualizada con el siguiente botón, que le presentará un fomulario el cual debe llenar con todos los datos. Una vez recibido por la administración procedera a actualizar sus datos en el sistema y le enviará un token por medio del email consignado en el formulario.

    Actualizar Datos

    Una vez recibido el token de acceso ingrese por usuario no registrado y complete el proceso de registro, ingresando un nombre de usuario y una clave, que debe guardar o memorizar pues serán las que le pide el sistema para poder registrarse cada vez que necesite utilizar las opciones de ingreso de información de ésta página

    Usuario No Registrado
    Vigilante/Staff no Registrado

    Si ya está registrado, ingrese por la opción Usuario Registrado, con su clave y usuario, esta clave no debe compartirla con nadie,para evitar ingresos indebidos que pueden alterar la información del conjunto
